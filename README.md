# Yokai Monitoring Dashboard

Тестовое задание: дашборд мониторинга ёкаев на Next.js 16 (App Router).

## Запуск локально

npm install
npm run dev

## Запуск в Docker

docker compose up --build

## Технологии

- Next.js 16 (App Router)
- React Query (@tanstack/react-query)
- react-hot-toast
- FSD-структура (app / entities / features / shared)
- SSE-имитация для real-time обновления уровней угроз
